# Interview Scheduler System

A full-stack Interview Scheduling application built using **Spring Boot** and **Angular**.

## 🔐 Features

* JWT-based Authentication
* Role-Based Authorization (ADMIN / USER)
* BCrypt Password Encryption
* Stateless Security using Spring Security
* RESTful APIs for managing interviews

## 🛠️ Tech Stack

* Backend: Spring Boot, Spring Security
* Frontend: Angular
* Database: MySQL
* Authentication: JWT

## 🔄 Flow

Login → Generate Token → Send Token → Validate → Role-based Access

## 📌 Key Highlights

* Implemented custom JWT filter for authentication
* Secured APIs using role-based access control
* Clean layered architecture (Controller, Service, Repository)
